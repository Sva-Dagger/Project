# 🏧 ATM Machine Simulator

A functional Python-based ATM interface that allows users to perform standard banking operations like balance inquiries, deposits, and withdrawals with secure PIN authentication.

## 🚀 Features
*   **Secure Login:** PIN-based authentication.
*   **Balance Inquiry:** Check real-time account balance.
*   **Cash Withdrawal:** Deduct funds with insufficient balance alerts.
*   **Deposits:** Add funds to the account instantly.
*   **Transaction History:** (Optional: list recent activity).

## 🛠️ Tech Stack
*   **Language:** Python 3.x
*   **Storage:** JSON or CSV (or simple variables for local state)

## 🔧 Installation & Setup
1. **Clone the repository:**
   ```bash
   git clone https://github.com/Sva-Dagger/Project.git
   cd Project
   ```

2. **Run the ATM Machine:**
   ```bash
   python TASK-01\ ATM\ MACHINE/atm_machine.py
   ```

## 📋 How to Use
- Enter your PIN to log in
- Select an operation (Balance, Withdrawal, Deposit)
- Follow the on-screen prompts
- Exit when done
